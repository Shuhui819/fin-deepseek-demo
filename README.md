# Financial Deepseek Demo

This project is a local Streamlit demo for AI-assisted financial analysis,
built on top of FinanceToolkit.

## Features
- Financial data analysis via FinanceToolkit
- Modular adapter-based design
- Local Streamlit web interface

## Environment
- Python 3.10
- Conda environment: fin-deepseek

## Run locally
```bash
conda activate fin-deepseek
streamlit run app/app.py
